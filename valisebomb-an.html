Project description

This project was commissioned by Stéphane Fardoux to create a transportable escape game

The goal was to create enigmas, and to integrate them in a playful and interactive way in a suitcase. The project was conducted over 2 terms by a group of 6 Adimaker students. During the first trimester, the puzzles were created and implemented in the suitcase. During the second trimester, we designed the suitcase, as well as fixed the bugs observed during the game and added new features.

Among the different puzzles, we have a walrus puzzle, an LED puzzle, an agility puzzle, and a code puzzle. To solve them, documents are available in the suitcase. The project was carried out with an Arduino board, LEDs, buttons, a magnetic sensor, a digital keypad, an LCD screen, a buzzer, a DF player and a speaker.
